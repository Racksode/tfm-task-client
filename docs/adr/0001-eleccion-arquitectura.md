# ADR 0001 - Elección de arquitectura

## Estado

Pendiente de decisión.

## Contexto

El proyecto debe definir una arquitectura adecuada para desarrollar un MVP funcional del TFM, sin sobredimensionar la solución.

## Decisión

Pendiente.

## Alternativas valoradas

- Monolito modular.
- Arquitectura separada frontend/backend.
- Arquitectura orientada a microservicios.
