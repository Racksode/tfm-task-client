## Resumen metodológico

El trabajo consistió en cerrar decisiones mínimas sobre stack tecnológico, persistencia, acceso a datos, autenticación, organización inicial y testing mínimo.

No se modificaron requisitos, modelo de datos, arquitectura, ADRs previos, README ni código. Tampoco se instalaron dependencias, se inicializó Next.js ni se crearon migraciones.
